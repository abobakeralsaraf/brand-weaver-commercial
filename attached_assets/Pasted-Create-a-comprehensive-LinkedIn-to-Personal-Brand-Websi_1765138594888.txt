Create a comprehensive LinkedIn to Personal Brand Website tool that extracts profile data and optionally builds a professional bilingual website.

PART 1: DATA EXTRACTION

Core Extraction Features:
- Personal information (name, headline, summary, location)
- Work experience (complete history with descriptions)
- Education (degrees, institutions, dates)
- Skills (all listed skills)
- Certifications (name, issuer, date, credential URL)
- Languages (language, proficiency level)

Visual Assets:
1. Profile Picture: Extract high-resolution version, convert to WebP format, generate alt text: "[Name] - [Headline]"
2. Header/Cover Image: Extract background banner image, convert to WebP format, optimize for web

Featured Posts (Embedded):
- Extract all featured/pinned posts from profile
- Get LinkedIn embed codes for each post
- Store post metadata: date, engagement, content preview
- Maximum 6 featured posts

Recommendations (Embedded):
- Extract all recommendations
- Get recommender's profile data: Name, Headline, Profile picture, LinkedIn profile URL (for clickable link)
- Get recommendation text
- Generate LinkedIn embed code for each recommendation
- Make recommender name/photo clickable to open their LinkedIn profile

Contact Information:
- Extract email (if available)
- Extract phone number (if available)
- LinkedIn profile URL
- Other social media links (if available)

Image Processing Pipeline - For ALL images:
1. Download original image
2. Convert to WebP format (quality: 85%)
3. Generate responsive versions: Mobile (max-width 640px), Tablet (max-width 1024px), Desktop (max-width 1920px)
4. Generate descriptive alt text using AI or structured format
5. Optimize file size (compress without quality loss)

Language Switcher JavaScript:
const langButtons = document.querySelectorAll('.lang-btn');
const htmlElement = document.documentElement;
langButtons.forEach(btn => {
  btn.addEventListener('click', () => {
    const lang = btn.dataset.lang;
    htmlElement.setAttribute('lang', lang);
    htmlElement.setAttribute('dir', lang === 'ar' ? 'rtl' : 'ltr');
    langButtons.forEach(b => b.classList.remove('active'));
    btn.classList.add('active');
    translatePage(lang);
    localStorage.setItem('preferredLanguage', lang);
  });
});

PART 8: ERROR HANDLING & EDGE CASES

Handle these scenarios gracefully:

1. Invalid LinkedIn URL: Show error with example URL
2. Private/Restricted Profile: Explain profile must be public with link to LinkedIn help
3. API Rate Limit Exceeded: Show countdown timer, offer to save partial data
4. No Featured Posts: Skip section or show message
5. No Recommendations: Hide section entirely
6. Missing Profile Picture: Generate placeholder with user's initials in selected color scheme
7. Missing Header Image: Use gradient background from selected colors
8. Image Download Failure: Retry 3 times, then use placeholder
9. Very Long Profile (100+ experiences): Show most recent 10, add "View Full Experience on LinkedIn" link
10. Special Characters in Name: Handle UTF-8 properly, test with Arabic/Chinese/emoji
11. Broken LinkedIn Embed: Fallback to static screenshot or show post text with "View on LinkedIn" link
12. Deployment Failure: Retry automatically (max 3 attempts), provide detailed error, offer ZIP download
13. GitHub Authentication Error: Clear instructions for creating personal access token
14. Slow API Response: Show loading animation, timeout at 60 seconds, offer cancel option
15. Incomplete Data Extraction: Save what was extracted, show warning, list missing sections
